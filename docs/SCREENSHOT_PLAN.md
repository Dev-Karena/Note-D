# NoteD — Screenshot Acquisition & Visual Presentation Strategy Plan

This document maps out the precise, professional screenshots recommended for portfolio reviews, store listings, and GitHub documentation for NoteD, engineered by **DEV KARENA**.

---

## 📸 Capture Strategy Guidelines

To present NoteD as a high-quality, production-ready product, follow these guidelines for each screenshot session:
*   **Uniform Status Bar:** Keep status bars clean. Use tools like Android's Demo Mode to show a 100% battery, active Wi-Fi (or no network signal to highlight the offline-first capability), and a professional system time (e.g., `10:00 AM`).
*   **Realistic Seed Data:** Populate cards with authentic, realistic content (e.g., *Refactoring Architecture plans*, *Local SQLite Database Indices*, *Shopping List*, *Productivity journal entries*). Avoid using mock text like `Lorem ipsum` or `test_note_1`.
*   **Framing & Overlays:** Present screenshots within high-resolution, bezel-less device mockups with consistent lighting.
*   **Consistent Themes:** Maintain a unified visual brand. Use clean Material 3 brand colors (deep indigo, royal blue, charcoal gray, slate backdrops) across screenshots.

## 🎨 Recommended Screenshot List & Mapping

### 1. Splash Screen
*   **Purpose:** Establishes the visual identity of NoteD on app launch.
*   **Data Displayed:**
    *   A centered NoteD logo (a minimalist stylized notebook icon with a subtle lock emblem).
    *   Sleek tagline text: *"Secure, Private, Local-First"*.
    *   A subtle loading indicator at the bottom.
*   **Presentation Strategy:** Capture the splash screen in both Light and Dark themes. Present them side-by-side inside high-resolution mockups to showcase visual consistency.

### 2. Lock / Authentication Screen
*   **Purpose:** Showcases NoteD's integrated security features on launch.
*   **Data Displayed:**
    *   Active visual lock shield statement: *"Vault Locked"*.
    *   A minimalist numeric decimal PIN pad layout (`0-9`, delete button, confirm check).
    *   An highlighted fingerprint icon prompting biometric logins.
    *   Haptic status indicator displaying: *"Scan fingerprint or enter PIN to edit"*.
*   **Presentation Strategy:** Show an active, glowing fingerprint scanning overlay. Add a clean, high-contrast drop-shadow behind the keypad elements.

### 3. Home Screen (Dashboard Workspace)
*   **Purpose:** Highlights NoteD's core interface, visual hierarchy, and category sorting.
*   **Data Displayed:**
    *   A clean search bar at the top with a search icon and custom category menu button.
    *   Horizontal category tag chips (e.g., *All notes*, *Work*, *Personal*, *Creative*, *Finance*) with active states.
    *   A *"Pinned"* notes section with 2 prioritized note cards styled with custom background colors.
    *   A list of recent notes arranged in a clean, dual-column staggered grid.
    *   An accessible Floating Action Button (FAB) at the bottom-right for creating notes.
*   **Presentation Strategy:** Present a professional landscape of custom notes. Show card backdrops in soft, high-contrast, modern colors (soft lavender, sage green, steel blue, pastel peach) with clear visual margins.

### 4. Create / Edit Note Screen
*   **Purpose:** Highlights NoteD's writing interface, categorization, and color-picking tools.
*   **Data Displayed:**
    *   Editable header field populated with professional text: *"System Architecture Design & Relational Tables"*.
    *   Active category selector dropdown showing *"Work"* selected.
    *   A rich markdown-styled text block with lists, bullet points, and code lines.
    *   A horizontal row of custom background card color circles with an active selection checkmark.
    *   An active alarm badge with a reminder time: *"Today, 4:30 PM (Daily)"*.
*   **Presentation Strategy:** Highlight an active cursor point within the body text and show one of the background color backdrops actively selected.

### 5. Global Search Screen
*   **Purpose:** Highlights the instantaneous, character-by-character local search capabilities.
*   **Data Displayed:**
    *   An active search input containing the search keyword *"Database"*.
    *   Search results highlighting occurrences of the word *"Database"* in gold or high-contrast styles.
    *   A filter tag at the top-right sorting matching items by date.
    *   A graceful placeholder illustration on empty query searches.
*   **Presentation Strategy:** Show matching notes filtered side-by-side with dynamic visual borders, highlighting how quickly NoteD filters results locally.

### 6. Active Reminder Setup Dialog
*   **Purpose:** Highlights NoteD's reminder configuration tools.
*   **Data Displayed:**
    *   An elegant overlay dialog box titled: *"Schedule Reminder"*.
    *   An interactive calendar date-picker showing dates.
    *   An interactive time-spinner selecting time (e.g., *11:30 AM*).
    *   A custom alarm repetition rate selector with *"Weekly"* selected.
    *   Action buttons: *"Cancel"* and *"Set Alarm"*.
*   **Presentation Strategy:** Capture the dialog as an elegant overlay, with the underlying write screen blurred slightly to direct focus to the alarm options.

### 7. Dark Mode Comparison View
*   **Purpose:** Showcases modern Material 3 Dark theme support.
*   **Data Displayed:**
    *   The complete Home Screen dashboard rendered in a high-contrast charcoal slate theme.
    *   System font tags rendered in balanced ivory-white with clear contrast.
    *   Soft dark colors applied to note cards to preserve readability in low-light environments.
*   **Presentation Strategy:** Capture the Light theme and Dark theme Home Screens side-by-side in high-resolution mockups to highlight the visual design of both modes.
